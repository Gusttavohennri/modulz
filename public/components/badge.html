Badge

A small marker for providing context and semantic meaning to parts of a UI—a pending application, a new feature, a warning message etc.

Default badge

Use a default badge to mark content which is notable but not particularly significant.

Default

Semantic badges

When content is more significant, you can apply a stronger semantic.

Badge with icon

You can add an icon to support the badge text and provide visual reinforcement. Use a 12dp icon with default badges and a 16dp icon with larger badges. The icon is wrapped in a div to provide the necessary margin.
